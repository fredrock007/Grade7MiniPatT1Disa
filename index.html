PAT 1: Mini-practical assessment task - Mechanical systems and control

Skill assessed:
Learners develop a working model of a hydraulic-syringe powered, linked-lever Security Gate using simple materials.
Scenario:
Disa Primary School is upgrading its entrance to improve learner safety and access control. Currently, the school gate is manually opened, which causes delays and safety risks during busy drop-off and collection times.
The school would like to install a low-cost automatic gate system that uses a hydraulic mechanism to open and close smoothly. The system must be safe, reliable, and easy to operate, even during load shedding. A manual override must be included so the gate can still work when there is no electricity.
The design must:
  • Open and close smoothly
  • Include a visible safety feature (colour marking or symbol)
  • Stop or slow when an obstruction is detected
  • Include a manual backup system
  • Be suitable for frequent daily use
Model Requirements:
  • Base: 40 cm × 20 cm
  • Gate: ±15 cm high and ±30 cm wide
  • Materials: Cardboard, plastic sheets, wooden sticks, syringes, tubing, rubber bands
  • Budget: Maximum R150
The gate must include levers or linkages to improve movement and control. The model must clearly show how a learner would safely operate it.
